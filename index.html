04 — PROJECT

About this build

DrowsyNet is a research prototype developed for an applied machine learning course project. It classifies open-eye, closed-eye, and yawning cues from facial crops using three CNN architectures, evaluated under a subject/sequence-aware leakage-safe split to avoid inflated accuracy from near-duplicate frames. This page is a static showcase of the trained models' results — the full training pipeline, Grad-CAM/SHAP analysis, and a leakage-safe evaluation protocol are documented in the accompanying research report and Kaggle notebook.

This is a visual drowsiness-cue classifier, not a clinical diagnosis tool. A single frame cannot confirm real drowsiness — a deployed system should aggregate predictions over a sliding window of frames and always keep a human driver in control.
